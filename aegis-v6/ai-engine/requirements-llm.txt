# AEGIS LLM Fine-Tuning Requirements
# ====================================
# Install separately from the main requirements.txt — these are only needed
# for LLM fine-tuning, dataset generation, and model evaluation.
#
# Quick install:
#   pip install -r requirements-llm.txt
#
# For CUDA 12.1 + torch 2.5 (RTX 2060, RTX 3xxx, RTX 4xxx):
#   pip install torch==2.5.1 torchvision==0.20.1 --index-url https://download.pytorch.org/whl/cu121
#   pip install -r requirements-llm.txt
#
# Windows WSL2 with Unsloth (2x faster training, recommended if available):
#   pip install unsloth
#   pip install -r requirements-llm.txt
#
# Hardware recommendations:
#   RTX 2060 (8GB)  → Llama 3.2 3B Q4  (safe) | Llama 3.1 8B Q4 (tight, batch=1)
#   RTX 3080 (10GB) → Llama 3.1 8B Q4  (comfortable)
#   RTX 4090 (24GB) → Llama 3.1 8B fp16 (fast, high quality)
#   Cloud A100-40G  → Llama 3.1 70B Q4 (max quality, ~$20 for 3 epochs)
